# 网络热词知识库

## YYDS（永远的神）

**含义**: YYDS是"永远的神"（Yong Yuan De Shen）的拼音首字母缩写，用来表达对某人或某事的极度赞美和崇拜。

**起源**: 这个词最早来源于电竞圈，粉丝用来形容职业选手Uzi的操作，后来逐渐在网络上流行开来。

**用法示例**:
- "这个选手的操作真是YYDS！"
- "这道菜的味道YYDS！"
- "这部电影YYDS，必看！"

**流行度**: 95/100

## emo

**含义**: emo是"emotional"的缩写，在网络用语中表示情绪化、抑郁、伤感的状态。

**起源**: 最初指代一种音乐风格和亚文化，后来演变成形容情绪低落的网络用语。

**用法示例**:
- "今天又emo了，不想说话"
- "深夜emo时刻"
- "听这首歌我emo了"

**流行度**: 88/100

## 绝绝子

**含义**: 表示非常好、非常棒的意思，是"绝了"的加强版，带有夸张和卖萌的语气。

**起源**: 由网络红人和短视频平台流行起来，特别在美妆、美食等领域使用频繁。

**用法示例**:
- "这个蛋糕的味道绝绝子！"
- "这个口红色号绝绝子！"
- "今天的妆容绝绝子！"

**流行度**: 82/100

## 内卷

**含义**: 指非理性的内部竞争，大家都在拼命努力但整体收益并未提升的现象。

**起源**: 源自学术概念"involution"，在2020年后在中国网络流行，用来描述各种过度竞争的社会现象。

**用法示例**:
- "现在的职场太内卷了"
- "不想内卷，只想躺平"
- "教育内卷化严重"

**流行度**: 90/100

---

## 破防

**含义**: 原指游戏中破除对方防御，网络用语中指心理防线被突破，情感上受到触动。

**起源**: 来源于游戏术语，后来被广泛用于形容情感上的触动和共鸣。

**用法示例**:
- "看到这段话我破防了"
- "这个场景直接破防"
- "被这个故事破防了"

**流行度**: 85/100
